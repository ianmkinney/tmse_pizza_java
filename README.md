# TMSE Pizza - Java Application

A JavaFX-based pizza ordering system application.

## Quick Start

### Build
```bash
./build.sh  # macOS/Linux
build.bat   # Windows
```

### Run
```bash
./run.sh    # macOS/Linux
```

## Prerequisites
- Java 21 or higher
- JavaFX 21 SDK

## Setup

See [SETUP_INSTRUCTIONS.md](SETUP_INSTRUCTIONS.md) for detailed setup instructions.

## Important Notes

- Application runs in full screen mode by default
- Default user accounts: customer/password123, admin/admin123
- All prices include automatic 8% tax calculation
- Maximum 4 toppings per pizza
- Orders are saved when confirmed
